### 时代隔离确认

```
次元放浪记 (艾莉尼) ─── 物理隔离 ─── 电脑世界篇 (贝尔泽ブト)
         │                                  │
         │                        ┌─────────┘
         │                        │ 泰斯塔蒙特夺舍米姆
         │                        ▼
         │                  新世界篇 (米姆)
         │                        │
         │                        │ (时代之后)
         │                        ▼
         └──── 不相关 ──── 地上真人篇 (拔示巴、蕾娜)
```

- **艾莉尼**与另外4个角色在时代和世界观上完全隔离
- **贝尔泽ブト**仅通过泰斯塔蒙特与米姆相连
- **拔示巴**和**蕾娜**同属地上真人篇，有直接的追捕关系
- **米姆**横跨电脑世界篇和新世界篇，但不知晓地上真人篇
