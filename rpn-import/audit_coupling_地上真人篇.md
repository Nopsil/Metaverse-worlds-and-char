## 0. 世界背景与时间框架

### 世界观
- 旧人类因污染自灭 → 机械种（神）管理地上 → 制造"真人"修复大地
- 旧人类意识遁入电子乐园"Metaverse/艾玛格"
- 真人分为**强硬派（革新者/INNOVATOR）**与**稳健派**
- 12名归还种从Metaverse再生于地上，引发争夺战

### 时间轴（两大战线）
| 阶段 | 时间标记 | 核心事件 |

| **战线A** | 归还种再生初期（~1年内） | 伊欧尼亚屠杀 → 艾菲斯沦陷 → 安特路亚废弃都市 → 塞雷基亚决战 → 亚哈战死 |

## 1. 重点验证：七项核心问题

### 1.1 伊欧尼亚屠杀——谁在场、谁死了

**在场角色：**
- **攻方（革新者）**：伊泽维尔（指挥官）、萨乌尔（金骑士/指挥官）
- **守方/被害方**：蕾娜（归还种）、约拿（卫士）、基德翁（卫士）、米莉安姆（艾菲斯增援卫士）、11名其他归还种、约拿母亲

**死亡清单：**
- **11名归还种**：被革新者"处理"（伊泽维尔本篇明确"已处理11名归还种"）
- **约拿母亲**：家被炮弹直击烧毁，葬身火海
- **众多伊欧尼亚卫士**：防线被突破，大量阵亡
- **萨乌尔半身**：被蕾娜"破晓者"首次启动的光束击毁半身（未死，后改造成机械体）

**关键因果链条：**
``
12名归还种再生 → 革新者伊欧尼亚奇袭 → 11人被杀 → 萨乌尔追蕾娜/基德翁 → 蕾娜破晓者首次启动 → 萨乌尔重伤 → 约拿/米莉安姆/基德翁/蕾娜趁乱乘联络艇逃离
``

**验证结论**：伊欧尼亚现场，除了存活逃离的蕾娜及其护卫3人，其余11名归还种全部死亡。萨乌尔在追击中受重伤但存活。伊泽维尔在森林中被米莉安姆狙击阻断，未能击杀最后一人。

### 1.2 拔示巴的母体改造链条

**完整链条（拔示巴本篇 Episodes 1-8）：**

1. **出生**："唯一自然分娩诞生的真人"——父亲亚哈的实验奇迹，生母死于产后
2. **养护设施**：与多名少年少女（同为拔示巴母亲细胞制成的"定制胎儿"）一同生活
3. **第一次实验（Ep2-3）**：被亚哈以"健康检查"名义麻醉后提取卵子，目睹同伴被改造为不成人形的肉块，好友恳求"杀了我"
4. **药物催熟（Ep5）**：强制服药 → 身体急速生长为母体（四肢、臀部、乳房增大）
5. **亚哈DNA植入（Ep6）**：卵子被注入亚哈遗传基因的胚胎 → 移回拔示巴体内 → 胚胎数周后脱落失败
6. **亚哈克隆体"仪式"（Ep7）**：与亚哈的克隆体交配 → 拔示巴掐死克隆体 → 咬碎后续克隆体的喉咙
7. **自杀阻止（Ep5）**：亚哈植入控制装置 → 拔示巴自杀即触发剧痛
8. **最终拒绝（Ep8）**：拔示巴恳求"大家一起死了就好" → 亚哈放弃强制实验 → 将拔示巴移至奥林匹亚斯圣堂为"圣母象征"

**验证结论**：源MD中母体改造流程连贯完整。关键转折在于Ep7中拔示巴持续掐死/咬碎克隆体导致亚哈不得不放弃实验计划。

### 1.3 伊泽维尔受拔示巴"祝福"的具体情节

**情节位置**：伊泽维尔.md Episode 9「圣女的祝福」

**具体描述**：
- 伊泽维尔曾是拔示巴的侍女，亲眼目睹拔示巴遭受的残酷实验后因恐惧辞职投身战场
- 伊泽维尔满脸颊有伤（与约拿交火留下），称其为"奇耻大辱的标记"
- 拔示巴听到后告白"我有点嫉妒了"
- **拔示巴用指甲刺入伊泽维尔脸颊的伤口，将伤口重新裂开**
- **拔示巴将沾满血液的手指伸入口中品味**
- **拔示巴强行掰开伊泽维尔的嘴，将自己的血液混入伊泽维尔口中**
- 拔示巴宣言："这是，我的**祝福**。尽情用这双手沾满鲜血，将一切毁灭吧——所有的一切"

**验证结论**：通过血液交换完成的"祝福"，本质是拔示巴对世界的诅咒的传染，而非传统意义上的赐福。伊泽维尔此后在战场上愈发疯狂——她咬断基德翁的脖颈吸血，最终在与基德翁/约拿的对决中战死。

### 1.4 蕾娜与梅尼的关系——梅尼是否"追上"蕾娜

**关系链条**：

| 时间点 | 事件 |
|-------|------|
| 蕾娜出发前 | 梅尼与蕾娜在神之庭园(GARDEN)勾小指约定"等梅尼到地上，蕾娜为她带路" |
| 蕾娜出发后 | 梅尼目送蕾娜进入"复苏之门" |
| 系统传唤梅尼 | 告知12人中仅剩蕾娜的生体反应，可能执行"净化"，梅尼以外交代理人身份前往地上 |
| 梅尼到达地上 | 在伊欧尼亚废墟遇到卫士纳荷姆 → 前往安特路亚殖民地追踪蕾娜足迹 |
| 安特路亚 | 通过都市监视系统确认蕾娜被某人拉着逃离，向东而去 |
| 塞雷基亚 | 遭遇拔示巴（白色女人）→ 被纳荷姆所救 → 拔示巴用枪形手指抵额 → 梅尼逃走 |
| 塞雷基亚郊外 | 发现亚哈的墓碑（由蕾娜/约拿所立），但梅尼未能见到蕾娜本人 |

**验证结论**：截至各MD结尾，**梅尼未能追上蕾娜**。梅尼追踪蕾娜的足迹（伊欧尼亚→安特路亚→塞雷基亚→东方），在塞雷基亚遭遇拔示巴后逃离，最近的接触距离是发现蕾娜留下的墓碑。塞雷基亚事件中梅尼遇到的白色女人正是拔示巴，梅尼直觉"决不能让这个女的和蕾娜姐见面"。

### 1.5 亚哈-布鲁对决与死亡

**完整对决链**（亚哈篇 Episodes 5-8 + 布鲁篇 Episodes 4-11）：

1. **佩尔修斯出征**：亚哈率革新者舰队进攻佩尔修斯，遭布鲁驾驶的机动兵器「米吉多·科古」拦截
2. **舰队战**：布鲁的机动兵器飞天→俯冲巨剑斩击，亚哈舰队大半沉没
3. **同归于尽**：亚哈旗舰与机动兵器相撞坠毁 → 亚哈被近卫兵用身体保护存活
4. **残骸相遇**：亚哈拖着即将死去的身体来到机动兵器残骸处，蓝色机械种布鲁现身
5. **嘴炮战**：布鲁说"赋予你们自我是错误，由我送你们上路是赎罪"。亚哈怒吼"把我们当成道具的创造主，还有资格说赎罪？"
6. **近身战**：布鲁空手vs亚哈长剑 → 布鲁优势（纯机械身体机能远超真人）→ 布鲁拳击亚哈腹部，掏出散弹枪"剑鱼"
7. **绝命逆转**：布鲁即将扣扳机时 → **一道激光穿过亚哈膝盖，将布鲁身体一分为二**
8. **亚哈补刀**：亚哈勉力起身，将长剑刺入布鲁残余的机械身体，确认其死亡
9. **亚哈追击**：亚哈乘搜索队留下的船，追入塞雷基亚都市

**最终对决（蕾娜篇 Episodes 8-10）**：
- 布鲁先被亚哈打倒（头颅被扔到皮阔特号甲板上）
- 亚哈拖着破败身体vs约拿/蕾娜
- **蕾娜用"破晓者"集中众人心意释放闪光，将亚哈连同其释放的光一同吞没消失**

**验证结论**：亚哈与布鲁的对决以**同归于尽**为第一阶段，亚哈亲手击杀布鲁，然后被蕾娜消灭。布鲁之前已启动机动兵器，在与亚哈舰队交战中严重受损。那道"穿膝激光"的来源在MD中未明确交代，可能是布鲁临死前的自我防御或其他第三方。

### 1.6 赛罗-凯南的身份关系

**来源**：凯南篇 Episodes 1-2

**核心设定**：

|- 赛罗·达文斯（Zero Davens）| ：拔示巴忠实的仆从，参与佩尔修斯殖民战
|- 拔示巴病倒后| ：赛罗做出决定——让拔示巴进入假死状态，并承诺"开拓通往愿望的道路"
|- 凯南·梅尔维亚斯| ：**赛罗的复制体/后继存在**，继承了赛罗的全部记忆
|- 关系本质| ：凯南的脑中持续回响着赛罗的记忆（"赛罗的记忆，仍在不停地回响着"）。凯南本人说"不是由赛罗·达文斯，而是我凯南·梅尔维亚斯来实现拔示巴的愿望"

**记忆共有机制**：凯南篇Ep2中凯南与镜中赛罗倒影对峙的描写，暗示两人不仅是复制关系，更有记忆/人格层面的重叠与争夺。

**验证结论**：赛罗=拔示巴忠实仆从（初代），凯南=赛罗的复制体/后继者（继承记忆但发展为独立人格）。凯南的目标同样是复活拔示巴。

### 1.7 泽法与拔示巴的克隆关系

**来源**：泽法篇 Episodes 2, 5

**核心设定**：

|- 泽法·尼亚德：拔示巴的**克隆素体之一**（复活的容器/复制品）
|- 制造目的| ：提取拔示巴生体情报+记忆覆盖到泽法身上，试图复活拔示巴
|- 结果| ：生体情报显现（泽法身体会成长，与普通真人不同），但意识是完全独立的存在
|- 其他幸存素体| ：蕾娅·艾·弗拉达（成为圣女/指导者，因"有指导者素养"被供奉）
|- 克隆关系网| ：泽法→拔示巴的肉体复制品；蕾娅→拔示巴的另一个复制品（意识独立）；沃特→蕾娅的"兄弟"，同为拔示巴复制品

**泽法被选为索罗私人教师的原因**：艾斯特尔说"没有比你更合适的人"——因为泽法是拔示巴的复制品，与拔示巴之子索罗有"母子"层面的适配性。

**验证结论**：泽法确为拔示巴的克隆复制品，但意识独立。她与拔示巴之间通过梦境有非物理连接——泽法反复梦见白发红眼女孩哭喊。
